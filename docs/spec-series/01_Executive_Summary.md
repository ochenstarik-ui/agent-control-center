# Документ 01. Executive Summary

> Это первый документ из серии архитектурной спецификации Agent Control
> Center.

## Назначение

Данный документ открывает полную архитектурную спецификацию проекта и
определяет: - цели продукта; - границы системы; - основные принципы; -
ключевые заинтересованные стороны; - критерии успеха; - обзор
архитектуры верхнего уровня.

## Правила серии

-   Каждый документ является самостоятельной частью общей спецификации.
-   Все документы имеют сквозную нумерацию.
-   После завершения серии документы могут быть объединены в единую
    спецификацию.
